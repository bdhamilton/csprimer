# Beep beep boop

Goal: write a program that, when running, beeps when you press a number key (that many times).

## Background

What makes this problem difficult is not the algorithm; the algorithm is very simple: `if input is a digit, beep that many times`. What makes this problem difficult is the technology. You can't do it in unless you know something about how ASCII works and how a terminal works.

For me, the key was to learn something new about the history of computing.
